# govm-windows

Simple Go Version Manager for Windows  
(no symlink, no admin, nvm-like UX)
---
## Install

```powershell
git clone https://github.com/yourname/govm-windows.git
cd govm-windows
.\install.ps1
```
Restart PowerShell after install.
---
## Usage
```
govm install 1.24.11
govm use 1.24.11
govm list
```
---
## Features
- No symlink
- No admin required
- PATH-based switching
- Windows native PowerShell
---
## Uninstall
```
.\uninstall.ps1
```
